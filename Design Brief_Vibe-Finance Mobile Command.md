# Design Brief: Vibe-Finance "Mobile Command"

**Archetype**: Industrial Utilitarian (Dark Mode)
**Stitch Prompt**: 
Create a mobile-optimized personal finance dashboard. 
- **Hero**: Large "Net Liquidity" total with a "Real Money" label.
- **Input**: A prominent "Add Transaction" button that opens a bottom drawer.
- **Numpad Focus**: The drawer must feature a large, custom-styled numeric keypad for rapid entry.
- **Visuals**: Use Soft Crimson (#ef4444) for CC balances and Neon Emerald (#10b981) for Cash.
