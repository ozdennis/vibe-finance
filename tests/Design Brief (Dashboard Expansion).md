# Design Brief: Vibe-Finance "Temporal & Wealth Command"

**Google Stitch Prompt**:
Update the mobile-first personal finance dashboard with the following sections:

1. **Global Header**: A sleek date picker showing "March 2026". Include left/right chevron arrows to jump between months.
2. **Dual-Hero Metric**: 
   - Primary (Large): "Net Liquidity" (Spendable cash).
   - Secondary (Smaller, subtle): "Total Net Worth" (Includes investments).
3. **The Investment Shelf**: A dedicated horizontal scrolling section strictly for "Investment" type accounts, showing MTD (Month-to-Date) growth.
4. **Reconciliation Action**: Inside the Account details view, a prominent "Sync Balance" button styled in Amber to indicate a corrective system action.
